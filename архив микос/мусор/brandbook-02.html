    О бренде

    Кодекс визуальной и смысловой идентичности театра клоунады «Микос»

    Кратко о бренде

    Экспериментальный театр, где клоунада становится языком глубокого диалога между сценой и зрителем.

    Цели бренда

    Создать узнаваемый культурный бренд с сильным визуальным кодом, привлечь новую интеллектуальную аудиторию.

    Задачи брендбука

    Единая система коммуникации: от афиш до соцсетей, сохранение характера «структурированного спектакля».

    Миссия

    Дарить зрителю опыт живого сопереживания через синтез клоунады, драмы и пластики. Смех, который остаётся в сердце.

    Ключевое сообщение

    «Искусство улыбаться всерьёз»

    Позиционирование

    Новаторский театр для думающей аудитории 20–45 лет, семьи, корпоративные клиенты.

    Задачи

    Рост узнаваемости на 40%, увеличение продаж билетов, формирование комьюнити вокруг бренда.

    Цветовые сочетания

    Разрешено

    Светлый фон + тёмный текст — базовое сочетание для основного контента.

    Разрешено

    Синий фон + светлый текст — для акцентных блоков и заголовков.

    Разрешено

    Красный акцент + белый текст — для кнопок и призывов к действию.

    Разрешено

    Золотой фон + тёмный текст — для премиум-материалов.

    Запрещено

    Красный + синий — конфликт акцентных цветов, снижает читаемость.

    Запрещено

    Светлый фон + золотой текст — низкий контраст, текст нечитаем.

    Запрещено

    Тёмный фон + красный текст — вибрация цветов, утомляет глаза.

    Запрещено

    Сторонние серые оттенки — не входят в палитру бренда.

    Палитра

    Основная: #F2F1F0 + #231F20
    Акцентные: #F43139 (драйв), #2C4068 (интеллект), #D4BF95 (теплота).

    Сочетания

    Тёмный текст на светлом, красный для кнопок, синий для фонов. Запрещены сторонние оттенки.
